## Introduction
The **Data Jobs Dashboard** is an interactive Power BI project that explores trends in the data job market.  
It provides insights into salaries, job demand, geographic distribution, and job characteristics across common data roles such as Data Analyst, Data Scientist, Data Engineer, and Machine Learning Engineer.

The goal of this dashboard is to transform raw job posting data into clear, actionable insights for job seekers, hiring managers, and anyone interested in the data industry.

---

## Key Questions Answered
- How do salaries differ across data-related roles?
- What is the median yearly and hourly pay in data jobs?
- Which platforms post the most data jobs?
- Where are data jobs located globally?
- How common are benefits such as health insurance, remote work, or degree requirements?

---

## Dashboard Features
- **Job Count Overview** with time-based trends
- **Median Salary Analysis** (yearly and hourly)
- **Role Comparison** by job volume and pay
- **Hourly vs Median Salary** comparison by role
- **Global Job Distribution** map
- **Drill-through by Job Title** for deeper role-specific insights
- **Job Characteristics Breakdown**, including:
  - Work-from-home availability
  - Degree requirements
  - Health insurance mentions
  - Employment type (full-time vs contract)

## Dashboard Overview

### Page 1: High-Level Market View

![Dashboard Page 1](/images/Project1_Page1.png)

This page acts as the **mission control** for the data job market. It provides a high-level overview of key metrics such as total job count, median yearly and hourly salaries, and overall salary ratings.

Key visuals on this page allow users to:
- Track **job posting trends over time**
- Compare **job counts across major data roles**
- Understand how **hourly pay compares to median yearly salaries**
- Quickly identify which roles dominate the market

This page is designed for fast insights and broad market awareness at a glance.

---

### Page 2: Role-Specific Deep Dive (Drill-Through)

![Dashboard Page 2](/images/Project1_Page2.png)

This page enables a **drill-through analysis by job title**, allowing users to explore individual data roles in greater detail.

It focuses on:
- **Salary distribution** (yearly and hourly)
- **Work-from-home availability**
- **Degree requirement mentions**
- **Health insurance mentions**
- **Global job distribution** for the selected role
- **Job platforms** with the highest number of listings
- **Employment type breakdown** (full-time vs contract)

This page is designed for deeper exploration, making it easy to compare roles and understand how job requirements and benefits differ across the data job landscape.
